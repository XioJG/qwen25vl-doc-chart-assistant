# Notes

本目录用于记录后续实验计划、问题排查和面试讲解草稿。

## 建议记录内容

- 单图推理测试记录。
- Gradio Demo 调试记录。
- JSON 输出失败案例和 prompt 调整思路。
- LoRA/QLoRA 微调计划和真实实验结果。
- 面试讲解提纲。

## 当前建议讲解主线

1. 项目定位：本地 Qwen2.5-VL-3B 多模态文档/图表理解系统。
2. 已完成能力：环境、仓库工程化、CLI、Gradio、prompt 模板、数据格式转换、基础评测。
3. 正在推进：真实脱敏样例推理、Demo 截图、JSON 输出稳定性。
4. 后续扩展：LoRA/QLoRA 微调、基线对比、更多公开脱敏样例。

## 记录边界

- 不记录 token、账号、私密路径或原始敏感数据。
- 不把计划中的指标写成已完成结果。
- 真实实验结果应写明环境、数据规模和命令。
